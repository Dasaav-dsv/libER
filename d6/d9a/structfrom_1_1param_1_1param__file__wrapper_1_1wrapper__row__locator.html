row_index_type from::param::param_file_wrapper::wrapper_row_locator::index

The entry index in the param table.

While a param's row index can be anything, the entry indices are assigned in the order of the param rows' appearance. So, param rows { 10001000, 10002000, 10002010 } will have indices { n, n + 1, n + 2 }, where n is the index of param row 10001000, provided there are no rows in-between.
